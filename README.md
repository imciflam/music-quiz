# Music quiz game

### Installation

1. Clone the repo
```sh
git clone  git://github.com/imciflam/guessmelody.git
```
2. Install NPM packages
```sh
npm install
```
3. Run the app
```sh
npm start
```
App will run on http://localhost:3502

3.1 to run both linter and mocha/chai tests with gulp, use
```sh
npm test
```
